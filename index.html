Tarea Programada 2 – Tetris

Implementación de una versión de Tetris en terminal, donde cada pieza tiene bloques de distintos color, y un sistema de puntos en base al color dominante de cada fila. La reglas incluyen combos, fila llenas y al crear una fila vertical del mismo color.

Tecnologías: Java
